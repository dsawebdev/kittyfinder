#kittyfinder

Installation Instructions

- Clone repository
```
git clone git@github.com:dsawebdev/kittyfinder.git
```
- CD into Directory 
```
cd kittyfinder <or whatever you named the clone repository> 
```
- Install dependecies
```
npm install
```
- Start local server
```
npm start
```
- Open up your prefered web-brower and view the app:
```
localhost:3000
```
